# INSTRUCCIONES DE ADMINISTRACIÓN - TRAGALERO

Este documento explica cómo gestionar el Directorio de Negocios y la Parrilla (Bento Grid) de Herramientas.

---

## 1. DIRECTORIO DE NEGOCIOS (restaurantapp.html)

El directorio de negocios (restaurantes, servicios, etc.) utiliza datos locales para una carga ultrarrápida.

### Cómo agregar o editar negocios:
1. Abre el archivo: `docs/assets/js/directory_data.js`
2. Verás una lista llamada `directoryData`. Cada objeto representa un negocio.
3. Puedes cambiar nombres, ciudades, CPs y categorías aquí.

---

## 2. CONFIGURACIÓN DE HERRAMIENTAS Y ENLACES (IMPORTANTE)

Para que el administrador no tenga que escribir enlaces manualmente, tú puedes pre-configurar las herramientas y sus links en el código.

### Cómo agregar, quitar o editar herramientas y sus links:
1. Abre el archivo: `docs/assets/js/tools_config.js`
2. Verás una lista llamada `TOOLS_CONFIG`. Cada objeto tiene un `name` (nombre) y un `url` (enlace):
   ```javascript
   { name: "Marketing kit", url: "https://tu-enlace-aqui.com" },
   ```
3. Si agregas una herramienta aquí, aparecerá automáticamente en el menú desplegable del editor.
4. Si cambias un enlace aquí, se aplicará a todos los cuadros que seleccionen esa herramienta en el futuro.

---

## 3. EDITOR DE PARRILLA / BENTO GRID (admin.html)

### Cómo usar el editor visual:
1. Abre `docs/admin.html` en tu navegador.
2. **Seleccionar Herramienta:** Al elegir una herramienta del menú desplegable, el sistema le asignará automáticamente el enlace que configuraste en `tools_config.js`.
3. **Tamaño y Color:** Puedes ajustar el tamaño (columnas/filas) y el color pastel de cada cuadro.
4. **Reordenar:** Arrastra los cuadros para cambiar su posición.
5. **GUARDAR:** Haz clic en **"Guardar Cambios"** para actualizar la base de datos de Supabase.

---

## 4. BASE DE DATOS (Supabase)

- **Tabla:** `bento_grid`
- Las credenciales están configuradas internamente en `docs/admin.html`.

---

¡Listo! Con esto el usuario final solo tiene que elegir el nombre de la herramienta y tú controlas los enlaces desde el archivo `.js`.
